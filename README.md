# 客户关系管理系统前端界面

基于 vue-admin-template


## Build Setup

```bash
# clone the project
git clone https://github.com/ZeroAC/CRM_web.git

# enter the project directory
cd CRM_web

# install dependency
npm install

# develop
npm run dev
```

This will automatically open http://localhost:9528

## Build

```bash
# build for test environment
npm run build:stage

# build for production environment
npm run build:prod
```
